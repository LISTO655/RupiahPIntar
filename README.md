# Template Website: Cinta Bangga Paham Rupiah

Template website edukasi Rupiah dengan struktur file yang sudah dirapikan agar mudah dikelola.

## Struktur Folder

```
CBP RUPIAH WEBSITE/
|-- index.html
|-- investasi.html
|-- compound-interest-calculator.html
|-- README.md
|-- assets/
|   `-- images/
|       |-- logo-white.png
|       |-- hero-banner.jpg
|       |-- about-section.jpg
|       |-- gallery-placeholder.png
|       `-- uang-rupiah-kertas-baru.jpg
|-- css/
|   |-- style.css
|   `-- pages/
|       |-- investasi.css
|       `-- compound-interest-calculator.css
|-- js/
|   `-- script.js
`-- docs/
    |-- FILE_INDEX.md
    |-- PLACEHOLDER.md
    `-- VISUAL_GUIDE.html
```

## Halaman Utama

- `index.html`: Halaman utama edukasi Rupiah.
- `investasi.html`: Halaman panduan menumbuhkan uang.
- `compound-interest-calculator.html`: Halaman kalkulator bunga majemuk.

## Path Statis Baru

- CSS global: `css/style.css`
- CSS halaman investasi: `css/pages/investasi.css`
- CSS halaman kalkulator: `css/pages/compound-interest-calculator.css`
- JavaScript global: `js/script.js`
- Seluruh gambar: `assets/images/*`

## Catatan Pengelolaan Konten

- Ubah teks utama langsung di file HTML halaman terkait.
- Jika ingin ubah gaya global, edit `css/style.css`.
- Jika ingin ubah gaya khusus halaman, edit file CSS di `css/pages/`.
- Dokumen panduan rinci tersedia di folder `docs/`.
