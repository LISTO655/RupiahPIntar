Visual Guide

Panduan visual ringkas untuk memahami struktur dan alur edit file.

Struktur Inti

Halaman utama tetap di root:

  • index.html
  • investasi.html
  • compound-interest-calculator.html

Aset dipusatkan di:

  • assets/images/
  • css/ dan css/pages/
  • js/script.js

Alur Edit Cepat

  1. Edit konten di file HTML.
  2. Edit tampilan global di css/style.css.
  3. Edit tampilan khusus halaman di css/pages/*.
  4. Update atau ganti gambar di assets/images/.

Catatan

Dokumen ini sudah disesuaikan dengan struktur folder terbaru.
